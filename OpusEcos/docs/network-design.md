# Opus Ecos Network Design

## Organization
- Name: OpusOrg
- MSP ID: OpusMSP

## Certificate Authority
- ca.opusecos.net

## Orderer
- orderer.opusecos.net

## Peers
- peer0.opusecos.net
- peer1.opusecos.net

## Channel
- opus-channel

## Initial Governance

The Opus Ecos network is initially operated by a single organization
(OpusOrg). Future organizations may join through governance policies
and certificate-based identity management.

## Security Goals

- Permissioned blockchain
- Certificate-based identities
- TLS encryption
- Role-based access control
- Auditability
